Hardware & Edge

Next-Gen
IoT Ecosystem.

When dealing with distributed hardware sensors, latency and data integrity are the hardest problems. I built an ecosystem from the PCB up to the cloud.
The system aggregates millions of data points a day using Rust on the edge and highly concurrent Go microservices in the cloud, all manageable via an intuitive frontend.

Role

Founder & Primary Architect

Tech Stack

C++ / Rust Golang NextJS TimescaleDB
Cloud Sync

The Problem

Existing sensor networks rely on heavily fragmented protocols. Manufacturers lock you into closed ecosystems preventing unified dashboarding. The latency was high, and provisioning new edge devices took days.

The Solution

I engineered a custom lightweight protocol over MQTT that auto-discovers nodes on the network. The backend seamlessly maps varying data structures into a unified Time-Series schema. As a result, deployment time went from days to literally bridging the device to power.
